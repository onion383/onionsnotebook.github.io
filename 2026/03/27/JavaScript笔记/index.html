事件 描述
onchange HTML 元素改变
onclick 用户点击 HTML 元素
onmouseover 鼠标指针移动到指定的元素上时发生

onkeydown 用户按下键盘按键
onload 浏览器已完成页面的加载

HTML DOM

  1. 概念 ：HTML DOM （文档对象模型），当浏览器在渲染网页的时候，浏览器会把标签拆成一个个元素进行渲染，JS可以通过一些方法改变这些元素

  2. HTML 元素的查找

    • 通过ID查找 使用 document.getElementById("ID名") var x=document.getElementById("intro");
    • 使用标签名查找 先用ID查找一个特殊的元素后（否则会很多，没有筛选效果），在下面查找特定的标签
    • 通过类名查找 document.getElementsByClassName("查找的类名") var x=document.getElementsByClassName("intro");

  3. 改变HTML

    • 直接在HTML写入
      document.write(你要写入的东西);
      document.getElementById("p1").innerHTML="新文本!";
    • 改变HTML元素的内容
      document.getElementById(你要更改的元素的ID).innerHTML=更改的内容
      var element=document.getElementById("header");
    • 改变HTML元素的属性
      document.getElementById(你要更改的元素是ID).属性名=新属性值
      document.getElementById("image").src="landscape.jpg";
    • 改变HTML样式（CSS)
      document.getElementById(你要更改的元素的ID).style.CSS的样式名字=新的样式值
      document.getElementById("p2").style.color="blue";
