前期基本准备

  • 新建文件夹，初始化项目

编程基本流程

很像一个店装修-开业-服务的过程

准备操作

  1. 引入Express框架 const 自定义框架名字 = require('express') //引入express框架 const express = require(‘express’)`

    • require 在node.js里面引入框架
    • const js 变量声明

  2. 创建一个服务对象 const 自定义服务对象名 = 自定义框架名字() //创造一个服务器对象叫做app const app = express()`

  3. 引入解析体

    • 基本格式 ：自定义服务对象名.use(express.解析体名(设置解析体内部参数)); app.use(express.json());//引入解析json的解析体`
    • 常见的解析体
      • 解析表单数据express.urlencoded()
        app.use(express.urlencoded({ extended: true }));
        extended 参数：
        - true：使用 qs 库解析，支持嵌套对象（如 user[name]=Tom）。
        - false：使用 Node 原生 querystring 库，仅支持简单键值对。
      • 解析json数据 ：`app.use(express.json());
      • 解析二进制数据：express.raw()
        // 解析二进制数据（默认仅处理 application/octet-stream 类型） app.use(express.raw({ type: 'application/octet-stream' }));
      • 解析纯文本 express.text()
        // 解析纯文本数据（默认仅处理 text/plain 类型）`` app.use(express.text({ type: ‘text/plain’ }));`

  1. 基本的响应和请求函数格式
  1. 响应请求的方法
方法 作用
get() 获取接受的数据
post() 发送处理后的数据
put() 修改/更新数据
detele() 删除数据
ues() 公用的方法
3. 接收和请求方法
  • 接收的方法 req
方法 作用 举例
req.query 获取 URL 问号?后的查询参数 console.log(req.query)
req.params 获取路由的动态参数
(指定ID的之u按查询)		 console.log(req.params)
req.body 获取前端提交的表单、Json 等参数 console.log(req.body)
req.cookies 获取前端的cookies —
req.get() 获取指定的http请求头 const token = req.get('Authorization') // 拿用户的浏览器/设备信息
const ua = req.get('User-Agent')
req.ip 获取访问者的IP —-
  • 发送的方法res
方法 作用 举例
res.send (' 内容 ') 给前端发送东西（文字） res.send('你好')
res.send('<h1>我是标题</h1>')
res.json({ ... }) 给前端发送json(作为接口) res.json({ name: "张三", age: 20 })
res.redirect ('/ 地址 ') 跳转到另一个地址 res.redirect('/home')
res.status(200) 设置返回代码 res.status(404).send('页面不存在')
res.cookie() / res.clearCookie() 删除浏览器储存的cookie —-
res.download() 给前端下载文件
res.sendFile() 给前端发送一个文件

  • 概念 ：类似于一个检查站，啥东西都要从我这里过
静态文件管理 express.static
  • 中间键注册
    自定义服务对象名.use('访问前缀', express.static('静态对象的文件夹名'));
    app.use('/public', express.static('public'));

  • 后续网页就可以使用静态文件了（图片、CSS）
路由模块化 express.Router()
